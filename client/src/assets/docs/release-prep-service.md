# Release Preparation

## Overview
Release Preparation helps Product Managers streamline the process of preparing artifacts for upcoming releases. It allows PMs to efficiently manage release notes, decisions, technical debt items, and artifacts to ensure a smooth release cycle.

## Key Capabilities
- Create and manage release prep sessions
- Retrieve and update release notes, decisions, and technical debt items
- Access available stories from various generators
- Export release notes, decision logs, and technical debt inventory in markdown format

## How to Use
To start using Release Preparation, begin by creating a new release prep session. Add relevant artifacts, such as release notes, decisions, and technical debt items. You can then access available stories for inclusion in the release. Finally, export the necessary documentation in markdown format for sharing with the team.

## Configuration & Fields
- **Release Prep Session Data**
  - **Purpose**: Stores information related to a specific release preparation session.
  - **Example**: Q3 Release Prep Session

- **User ID**
  - **Purpose**: Identifies the user associated with the session.
  - **Example**: John Doe

- **Include Released Stories**
  - **Purpose**: Determines whether already released stories should be included.
  - **Example**: True

- **Artifact ID**
  - **Purpose**: Identifies a specific artifact for management.
  - **Example**: 1234

- **Session ID**
  - **Purpose**: Unique identifier for a release preparation session.
  - **Example**: RP2023-001
