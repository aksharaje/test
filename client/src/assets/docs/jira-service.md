# Jira Manager

## Overview
Jira Manager helps you streamline project management tasks and collaboration within your team. Easily track progress, assign tasks, and monitor project milestones all in one place.

## Key Capabilities
- Create and assign tasks
- Monitor project progress
- Track project milestones
- Collaborate with team members

## How to Use
To get started with Jira Manager, begin by creating a new project and assigning tasks to team members. Monitor progress and track milestones to ensure timely completion of projects. Use the collaboration features to communicate with team members and keep everyone aligned.

## Configuration & Fields
- **Return URL**: Redirect users to a specific page after completing an action.
  - **What it's for**: Specify where users should be directed after completing a task.
  - **Example**: After task completion, redirect users to a feedback survey.
  
- **Session**: Manage user authentication and access control.
  - **What it's for**: Control user access to specific project information.
  - **Example**: Grant project managers full access while team members have restricted access.
