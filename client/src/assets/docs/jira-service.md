# Jira Manager

## Overview
Jira Manager helps you streamline project management tasks by organizing and tracking project details efficiently. It enables you to oversee project progress, collaborate with team members, and ensure timely delivery of projects.

## Key Capabilities
- Manage project states and ensure timely completion
- Generate OAuth URLs for secure integrations
- List and manage project integrations

## How to Use
To make the most of Jira Manager, start by cleaning up expired project states to maintain data accuracy. Then, generate OAuth URLs for seamless integrations with external tools. Finally, list and manage project integrations to keep all project information in one place.

## Configuration & Fields
- **Return URL**: URL to redirect users after OAuth authentication
  - **What it's for**: Specifies where users should be redirected post-authentication
  - **Example**: "https://www.example.com/dashboard"
  
- **Session**: Active work session for accessing project integration details
  - **What it's for**: Provides access to project-specific information during a work session
  - **Example**: "Team collaboration meeting"
