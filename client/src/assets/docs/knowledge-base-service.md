# Knowledge Base Manager

## Overview
The Knowledge Base Manager helps Product Managers organize and access information essential for decision-making and project success. It enables efficient knowledge sharing and retrieval within the team.

## Key Capabilities
- Create and manage knowledge bases
- Add and index documents for easy retrieval
- Search for information within knowledge bases
- Collaborate by adding external sources like GitHub repositories

## How to Use
To get started, create a new knowledge base and populate it with relevant documents. Index these documents to enable quick search and retrieval. Collaborate with team members by adding external sources for comprehensive knowledge management.

## Configuration & Fields
- **Connected Knowledge Base**
  - **Purpose**: Identifies the knowledge base to perform actions on.
  - **Example**: "Product Launch Strategies"
  
- **Owner**
  - **Purpose**: Specifies the owner or user associated with the knowledge base.
  - **Example**: "John Doe"
  
- **Knowledge Base Data**
  - **Purpose**: Contains information to create or update a knowledge base.
  - **Example**: "Detailed market analysis report"
  
- **Update Data**
  - **Purpose**: Provides details for updating a knowledge base.
  - **Example**: "Updated financial projections"
  
- **Document**
  - **Purpose**: Represents a piece of content added to a knowledge base.
  - **Example**: "Competitor analysis presentation"
  
- **URL**
  - **Purpose**: Specifies the link to an external source, like a GitHub repository.
  - **Example**: "https://github.com/organization/project"
  
- **Query**
  - **Purpose**: Defines the search term to look for within a knowledge base.
  - **Example**: "Market trends 2022"
  
- **Limit**
  - **Purpose**: Sets the maximum number of search results to display.
  - **Example**: "10"
  
- **Threshold**
  - **Purpose**: Establishes the minimum relevance score for search results.
  - **Example**: "0.8"
