# Knowledge Base Manager

## Overview
The Knowledge Base Manager helps Product Managers organize and access essential information efficiently. It enables PMs to centralize knowledge, streamline collaboration, and enhance decision-making processes.

## Key Capabilities
- Create and manage knowledge bases
- Add and index documents for quick retrieval
- Collaborate by sharing external sources like GitHub repositories
- Search for information within knowledge bases

## How to Use
To get started with the Knowledge Base Manager, begin by creating a new knowledge base and adding relevant documents. Index the documents for easy searchability and collaborate with team members by sharing external sources. Use the search feature to quickly locate information within your knowledge bases.

## Configuration & Fields
- **Connected Knowledge Base**
  - **Purpose**: Identifies the specific knowledge base being accessed.
  - **Example**: "Product Launch Strategy"

- **Document**
  - **Purpose**: Represents a file or piece of content added to the knowledge base.
  - **Example**: "Market Research Report"

- **URL**
  - **Purpose**: Links an external GitHub source to the knowledge base.
  - **Example**: "https://github.com/yourrepository"

- **Query**
  - **Purpose**: Specifies the search term to find information within the knowledge base.
  - **Example**: "Competitor Analysis"

- **Limit**
  - **Purpose**: Sets the maximum number of search results to display.
  - **Example**: 10

- **Threshold**
  - **Purpose**: Adjusts the relevance threshold for search results.
  - **Example**: 0.8
