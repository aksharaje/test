# Code Chat

## Overview
Code Chat helps Product Managers collaborate with their team members in real-time, fostering knowledge sharing and enhancing project efficiency.

## Key Capabilities
- Create and manage work sessions
- Access code knowledge bases
- Send messages within sessions

## How to Use
To begin, create a new work session to collaborate with team members. Access code knowledge bases to gather information and insights. Communicate effectively by sending messages within sessions to keep everyone aligned.

## Configuration & Fields
- **Connected Knowledge Base**
  - **Purpose**: Select the knowledge base related to the project for reference.
  - **Example**: "Java Best Practices"

- **Owner**
  - **Purpose**: Identify the owner or user responsible for the session.
  - **Example**: John Doe

- **Session Reference**
  - **Purpose**: Reference the specific work session for tracking and continuity.
  - **Example**: Session #12345
