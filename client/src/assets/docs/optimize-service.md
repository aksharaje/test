# Optimized Flow Manager

## Overview
The Optimized Flow Manager helps Product Managers enhance and refine their content strategies by analyzing and optimizing user flows. It enables PMs to make data-driven decisions to improve user engagement and conversion rates.

## Key Capabilities
- View all flows with performance statistics
- Access detailed insights for individual flows
- Collect feedback on specific flows
- Save optimized prompts for better user interactions
- Activate different versions of flows for testing
- Create split tests to compare flow variations

## How to Use
To start optimizing your flows, begin by reviewing all flows to identify performance trends. Dive deeper into specific flows to understand user interactions and gather feedback. Save optimized prompts based on insights gained and activate different versions for testing purposes. Lastly, create split tests to compare the effectiveness of different flow variations.

## Configuration & Fields
- **Session**: A work session within the application
  - **What it's for**: Represents your current working session to access flow data
  - **Example**: Morning brainstorming session with the team
- **Flow ID**: Identifier for a specific flow
  - **What it's for**: Identifies the unique flow you want to analyze or optimize
  - **Example**: Onboarding flow for new users
- **Optimized Prompt**: Enhanced message for user interaction
  - **What it's for**: Improves user engagement by providing optimized content
  - **Example**: Tailored welcome message for improved user onboarding
- **Version ID**: Numeric identifier for different flow versions
  - **What it's for**: Represents different variations of a flow for testing purposes
  - **Example**: Version 1 for the original flow design
