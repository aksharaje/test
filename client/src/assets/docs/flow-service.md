# Flow Manager

## Overview
The Flow Manager helps Product Managers streamline and manage their project workflows efficiently. It enables PMs to oversee the progress of their tasks, collaborate with teams, and ensure project milestones are met on time.

## Key Capabilities
- Create new project flows
- Update existing project details
- Track project executions
- Start project tasks
- Retrieve project information

## How to Use
To get started with the Flow Manager, begin by creating a new project flow. Add relevant details and milestones to keep track of your project progress. Update the flow as needed and start project tasks to initiate project execution. Use the manager to retrieve project information and monitor the status of each task.

## Configuration & Fields
- **Session**: A session representing your current work environment.
  - **What it's for**: Helps identify and authenticate your access to project data.
  - **Example**: Logging in to the application to view and manage project flows.

- **ID**: Unique identifier for a specific project flow or execution.
  - **What it's for**: Enables precise referencing and retrieval of project details.
  - **Example**: Using the ID to locate and update a particular project flow.

- **Data**: Information related to a project flow or task.
  - **What it's for**: Stores project-specific details and updates.
  - **Example**: Adding new milestones or updating progress status within a project flow.

- **Context**: Additional context or information for starting project executions.
  - **What it's for**: Provides necessary details for initiating project tasks effectively.
  - **Example**: Including specific instructions or requirements when starting a project task.
