
# Measurement Framework Manager

## Overview
The Measurement Framework Manager helps Product Managers efficiently create and manage measurement frameworks for their projects. By using this tool, PMs can streamline the process of setting up and tracking key metrics, data sources, and dashboards, ultimately enabling them to make data-driven decisions and monitor project progress effectively.

## Key Capabilities
- Create new measurement framework sessions
- Retrieve and analyze session metrics
- Manage data sources for sessions
- Generate measurement frameworks using AI
- Retry failed sessions for continuous improvement

## How to Use
To get started with the Measurement Framework Manager, begin by creating a new measurement framework session. Input the necessary data sources and metrics for your project. Utilize the AI-powered framework generation feature to enhance your measurement process. Analyze session metrics and review dashboards to gain insights into your project's performance. In case of a failed session, use the retry function to make necessary adjustments and improve outcomes.

## Configuration & Fields
- **Measurement Framework Session Create**
  - **Measurement Framework Session**: The main session data for your project, including metrics and data sources.
  - **Example**: Define the key metrics and data sources for your new project session.
  
- **Session Reference**
  - **Purpose**: Identifies the specific session you want to access or modify.
  - **Example**: Enter the session reference number to retrieve detailed information.
  
- **Connected Knowledge Base**
  - **Purpose**: Links your project to relevant knowledge bases for additional insights.
  - **Example**: Connect your project to the company's knowledge base for comprehensive analysis.

- **Owner**
  - **Purpose**: Designates the owner or user responsible for the session.
  - **Example**: Assign yourself or a team member as the owner of the session.

- **Retry a Failed Session**
  - **Purpose**: Allows you to reattempt a session that encountered issues.
  - **Example**: Retry a failed session to analyze and rectify any errors.
