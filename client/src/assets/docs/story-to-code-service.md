# Story to Code Converter

## Overview
The Story to Code Converter helps Product Managers transform user stories into code efficiently, streamlining the development process and ensuring alignment between requirements and implementation.

## Key Capabilities
- Create new story-to-code sessions
- Retrieve and manage code generation sessions
- List and select user story artifacts
- Generate code from user stories
- Convert generated files into a ZIP file

## How to Use
To start using the Story to Code Converter, begin by creating a new session where you can input the description, source, and relevant details. Once the session is created, you can process it to generate the code. You can also manage existing sessions, retrieve artifacts, and convert the generated code into a downloadable ZIP file.

## Configuration & Fields
- **Title**: Create a new story-to-code session
  - **Purpose**: Initiates a new code generation session based on user story inputs.
  - **Example**: "Develop a login feature with social media integration"

- **Input Description**: Description of the user story
  - **Purpose**: Provides details about the user story for code generation.
  - **Example**: "As a user, I want to log in using my Google account."

- **Input Source**: Source of the user story
  - **Purpose**: Specifies where the user story originated from.
  - **Example**: "Product Backlog Item #123"

- **Source Artifact**: ID of the source artifact
  - **Purpose**: Links the user story to a specific artifact for reference.
  - **Example**: "Epic #456"

- **Tech Stack**: Technology stack to be used for code generation
  - **Purpose**: Specifies the technology environment for the code.
  - **Example**: "React, Node.js"

- **Connected Knowledge Base**: Knowledge bases with relevant code content
  - **Purpose**: Provides access to existing code resources for reference.
  - **Example**: "Frontend Development Best Practices"

- **Owner**: User responsible for the session
  - **Purpose**: Assigns ownership of the code generation session.
  - **Example**: "John Doe"
