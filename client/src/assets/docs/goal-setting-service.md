
# Goal Setting Manager

## Overview
The Goal Setting Manager helps Product Managers efficiently create and manage goal setting sessions, enabling them to track progress and drive team alignment towards objectives.

## Key Capabilities
- Create new goal setting sessions
- View and manage existing sessions
- Generate goals using AI
- Retry failed sessions for improved outcomes

## How to Use
To begin, start by creating a new goal setting session. Define the session details, such as the session name, objectives, and participants. Once the session is created, you can access and manage it easily through the interface. Use the AI-powered goal generation feature to enhance goal setting efficiency. In case of any failed sessions, utilize the retry functionality to optimize results.

## Configuration & Fields
- **Session Name**: Name your goal setting session for easy reference.
  - **Purpose**: Identifies the session for quick recognition.
  - **Example**: Q3 Sales Strategy Planning

- **Objectives**: Define the key goals or outcomes expected from the session.
  - **Purpose**: Clarifies the focus and desired results.
  - **Example**: Increase customer retention by 10%.

- **Participants**: List the team members or stakeholders involved in the session.
  - **Purpose**: Ensures team alignment and collaboration.
  - **Example**: John Doe, Marketing Manager

- **Goal Setting Session Reference**: Unique identifier for each session.
  - **Purpose**: Links actions and data to the specific session.
  - **Example**: GS2023-005
