# Library Management

## Overview
The Library Management tool helps Product Managers efficiently manage their library of resources, enabling them to easily access, organize, and analyze relevant information to make informed decisions and drive product innovation.

## Key Capabilities
- Create new books
- List available books
- Retrieve book details
- Generate content for books
- Delete books

## How to Use
To get started with Library Management, begin by listing the available books to review existing resources. You can then create new books to organize and store relevant information. When needed, retrieve specific book details to delve deeper into the content. Utilize the content generation feature to streamline the process of creating book content. Finally, if necessary, delete books that are no longer needed to maintain a tidy library.

## Configuration & Fields
- **Connected Knowledge Base**
  - **Purpose**: Identifies the knowledge base associated with the book.
  - **Example**: "Market Research Reports"

- **Owner**
  - **Purpose**: Indicates the user who owns or is responsible for the book.
  - **Example**: "John Doe"

- **Commit Hash**
  - **Purpose**: Tracks the version of changes made to the book content.
  - **Example**: "a1b2c3"

- **Change Summary**
  - **Purpose**: Provides a brief overview of the modifications made to the book.
  - **Example**: "Updated product features based on user feedback"
