# Library Manager

## Overview
The Library Manager application helps Product Managers efficiently manage their library of resources, enabling them to easily access, organize, and collaborate on content related to their projects.

## Key Capabilities
- Create new books in the library
- List available books and their details
- Retrieve specific book information
- Generate content for a book
- Delete unnecessary books

## How to Use
To get started with Library Manager, begin by creating a new book in the system. You can then list all available books, retrieve specific details of a book, generate content for a book, and delete any books that are no longer needed.

## Configuration & Fields
- **Session**: A session represents your current work session within the application.
  - **What it's for**: Helps track your activity and maintain security.
  - **Example**: Your current work session where you are managing book content.
  
- **User ID**: Your unique identification in the system.
  - **What it's for**: Allows for personalized access and actions within the application.
  - **Example**: Assigning a book creation to a specific team member.

- **Book ID**: The unique identifier for each book in the library.
  - **What it's for**: Helps differentiate between different books and retrieve specific information.
  - **Example**: Identifying the book you want to generate content for.

- **Knowledge Base ID**: Identifies the knowledge base associated with a book.
  - **What it's for**: Links the book to relevant knowledge sources.
  - **Example**: Connecting a book on marketing strategies to the marketing knowledge base.

- **Commit Hash**: A unique identifier for a specific version of content.
  - **What it's for**: Tracks changes and versions of the book content.
  - **Example**: Recording the latest update made to a book's content.

- **Change Summary**: A brief description of the changes made to the book content.
  - **What it's for**: Provides context for the modifications made.
  - **Example**: Summarizing the edits made to a chapter in the book.
