
# Story Generator

## Overview
The Story Generator tool helps Product Managers create compelling narratives for their products, enhancing communication and alignment across teams. It enables PMs to craft engaging stories to convey product vision, strategy, and value propositions effectively.

## Key Capabilities
- Generate narrative prompts
- Manage knowledge base context
- Create and update artifacts
- List and retrieve artifacts

## How to Use
To start, begin by generating narrative prompts tailored to your specific product or feature. Use the knowledge base context to enrich your stories with relevant information. You can then create, update, or delete artifacts to refine your narratives. Lastly, list and retrieve artifacts for easy access and sharing.

## Configuration & Fields
- **Type**: Type of prompt
  - **What it's for**: Select the type of narrative prompt you want to generate.
  - **Example**: Select "Product Launch" for crafting a story around a new product introduction.

- **Title**: Title of the prompt
  - **What it's for**: Provide a title for your narrative prompt to give context.
  - **Example**: Enter "Innovative Features Showcase" as the title for a feature highlight story.

- **Session**: Work session context
  - **What it's for**: Define the work session context for generating or managing narratives.
  - **Example**: Choose a specific project session to generate a story aligned with that project.

- **Request**: Data for generating a narrative
  - **What it's for**: Input the necessary data to create a narrative based on the provided request.
  - **Example**: Input customer feedback and product insights to generate a customer success story.

- **User ID**: User identification
  - **What it's for**: Specify the user for whom artifacts are being listed.
  - **Example**: Enter the user ID to retrieve artifacts specific to a particular team member.
