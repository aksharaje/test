# Feedback Manager

## Overview
The Feedback Manager helps Product Managers gather and analyze feedback from users to drive product improvements and enhance user satisfaction.

## Key Capabilities
- Create feedback entries
- List and review feedback entries

## How to Use
To start using the Feedback Manager, begin by creating feedback entries based on user input. Once feedback is collected, you can easily list and review the entries to identify trends and prioritize product enhancements.

## Configuration & Fields
- **Session**: Represents the work session during which feedback was collected.
  - **Purpose**: Helps organize feedback based on the context in which it was provided.
  - **Example**: "Product Testing Session - June 2021"
  
- **Data**: Contains the feedback information provided by users.
  - **Purpose**: Captures user suggestions, comments, or issues for product improvement.
  - **Example**: "Suggested adding a chat feature for better user interaction"
