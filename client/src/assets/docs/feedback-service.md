# Feedback Manager

## Overview
The Feedback Manager allows Product Managers to gather and analyze feedback from users, enabling them to make data-driven decisions to improve their products and services.

## Key Capabilities
- Create feedback entries
- List feedback entries

## How to Use
To start using the Feedback Manager, begin by creating new feedback entries to capture user insights. You can then list all feedback entries to review and analyze the collected data.

## Configuration & Fields
- **Session**: Represents the work session during which feedback was collected.
  - **What it's for**: Identifies the context in which feedback was provided.
  - **Example**: Weekly product demo session.

- **Data**: Contains the feedback information provided by users.
  - **What it's for**: Captures user comments, suggestions, or issues.
  - **Example**: User's request for a new feature implementation.
