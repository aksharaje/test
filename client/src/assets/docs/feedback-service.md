
# Feedback Management

## Overview
Feedback Management helps Product Managers gather and analyze feedback from users to improve products and services. It enables PMs to make data-driven decisions and enhance user satisfaction.

## Key Capabilities
- Create feedback entries
- List and view existing feedback

## How to Use
To start using Feedback Management, begin by creating feedback entries based on user input. You can then list and review the feedback to identify trends and insights for product enhancement.

## Configuration & Fields
- **Session**: Represents the work session during which feedback is collected.
  - **What it's for**: Helps organize feedback based on different work sessions or events.
  - **Example**: "Weekly user testing session on new feature rollout."
  
- **Data**: Contains the feedback details provided by users.
  - **What it's for**: Captures user opinions, suggestions, or issues.
  - **Example**: "User A reported a bug in the checkout process."
